Kadang saat membuka Chrome, muncul BSOD (Blue Screen of Death) yang bikin laptop restart. Penyebab utamanya sering karena fitur Hardware Acceleration bentrok dengan driver GPU.

ðŸ›‘ Penyebab Utama

  • Bug pada driver GPU (Intel / NVIDIA / AMD).
  • Fitur Hardware Acceleration aktif di Chrome.
  • Registry belum diset untuk override setting Chrome.

âœ… Solusi Praktis

1. Nonaktifkan Hardware Acceleration lewat Chrome

  1. Buka chrome://settings/system
  2. Matikan opsi Use hardware acceleration when available
  3. Restart Chrome

2. Pakai Registry untuk Toggle

Buka regedit lalu masuk ke:

HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Google\Chrome

Buat atau ubah key berikut:

Nama: HardwareAccelerationModeEnabled
Tipe: REG_DWORD
Nilai: 0 (nonaktif), 1 (aktif)

3. Skrip Batch Otomatis

@echo off
echo === Toggle Hardware Acceleration Chrome ===
echo 1. Nonaktifkan
echo 2. Aktifkan
echo 3. Hapus Setting
set /p choice="Pilih opsi (1/2/3): "

if "%choice%"=="1" (
    reg add "HKLM\SOFTWARE\Policies\Google\Chrome" /v HardwareAccelerationModeEnabled /t REG_DWORD /d 0 /f
    echo Hardware Acceleration DIMATIKAN.
    goto end
)

if "%choice%"=="2" (
    reg add "HKLM\SOFTWARE\Policies\Google\Chrome" /v HardwareAccelerationModeEnabled /t REG_DWORD /d 1 /f
    echo Hardware Acceleration DIAKTIFKAN.
    goto end
)

if "%choice%"=="3" (
    reg delete "HKLM\SOFTWARE\Policies\Google\Chrome" /v HardwareAccelerationModeEnabled /f
    echo Setting dihapus, Chrome kembali default.
    goto end
)

echo Pilihan tidak valid.

:end
pause
