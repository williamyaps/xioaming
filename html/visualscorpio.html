?? SHIO: ULAR API (Fire Snake) – Tahun 1977

Tahun 1977 berelemen Api Yin, hewan Ular. Ini bukan ular biasa—paling bijak dan paling intens. “The wisest and most intense snake.”

Sifat Dasar Shio Ular:

  • Bijak, intuitif, dalam. Radar alami membaca orang dan situasi.
  • Ambigu, misterius, charming. Susah ditebak isi pikiran.
  • Determinasi tinggi tapi melingkar dulu baru menyambar.
  • Elemen Api memberi keberanian, gairah, dan karisma magnetis yang sedikit menakutkan.

Shio Ular Api disebut “Double Fire” oleh Gemini: Api Yin redup tapi sangat panas dan fokus, seperti laser.

Kecocokan Shio

  • Paling cocok: Kerbau (Ox), Ayam (Rooster), Naga (Dragon).
  • Kurang cocok: Babi (Pig), Monyet (Monkey).

Stabilitas Kerbau dan komunikasi intelektual Ayam sangat mendukung jiwa investigatifmu.

?? ZODIAK: SCORPIO (Matahari 4° Scorpio)

Matahari di Scorpio, elemen air berelemen tetap. Penguasa: Pluto (transformasi) dan Mars (agresivitas).

  • Intens, passionate, all or nothing.
  • Loyalitas ekstrem; pengkhianat akan langsung dihapus.
  • Magnet misterius, penjaga rahasia ulung.
  • Butuh kontrol dan terdorong menyelami misteri kehidupan.

?? BULAN DI GEMINI (sekitar 3° Gemini)

Emosi dan inner child. Dualitas besar dengan Scorpio Sun.

  • Otak butuh stimulus; kalau bosan mood bisa hancur.
  • Ekspresi emosi lewat kata-kata dan analisis—merasa dengan berpikir.
  • Perubahan emosi cepat, dari dalam tiba-tiba enteng.
  • Butuh komunikasi sebagai katup emosi.
Gemini menyebut ini aspek Quincunx (150°) dengan Matahari—titik konflik internal terbesar: hati ingin privasi (Scorpio), emosi ingin update data (Gemini).

?? ASCENDANT: CANCER (26° Cancer Rising)

Topeng sosial. Orang melihatmu sebagai sosok hangat, peka, dan melindungi.

  • Aura keibuan/bapak pelindung.
  • Sensitif, mood dipengaruhi lingkungan dan kenangan.
  • Bulan penguasa Ascendant ? Bulan Gemini makin kuat, menavigasi dunia lewat kepekaan plus komunikasi.
